O escopo do fiscal-rs é EXATAMENTE o mesmo do ACBr. Todos os módulos são relevantes: NFe, CTe, MDFe, NFSe, SAT, SPED, boletos, PIX, TEF, serial, etc.

**Why:** O usuário corrigiu a suposição de que apenas NFe/DFe seria relevante. O projeto não é parcial — é uma reimplementação completa em Rust.

**How to apply:** Ao explorar o ACBr como referência (`.reference/acbr/`), nunca descartar módulos como "fora do escopo". Tudo pode ser consultado e é pertinente.
